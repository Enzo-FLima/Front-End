💻 Redes de Computadores

Introdução às Redes de Computadores

Em um mundo hiperconectado, entender como os dados viajam de um ponto A para um ponto B é fundamental para qualquer entusiasta de tecnologia ou aspirante a desenvolvedor.

1)O que é uma rede de computadores

Em termos simples, uma rede é um conjunto de dispositivos (nós) conectados entre si para compartilhar recursos e trocar informações. Esses recursos podem ser arquivos, impressoras ou a própria conexão com a internet.

Componentes basicos:

  • Dispositivos Finais: Computadores, smartphones, servidores e IoT.
  • Meios de Transmissão: Cabos de fibra óptica, cabos de cobre (Ethernet) ou ondas de rádio (Wi-Fi).
  • Dispositivos de Interconexão: Roteadores, switches e access points.

2. Classificação das Redes

As redes são classificadas principalmente pela área geográfica que ocupam:

Tipo Nome por extenso Descrição
PAN Personal Area Network Redes pessoais (ex: Bluetooth entre celular e fone).
LAN Local Área Network Redes locais (sua casa, escola ou escritório).
WAN Metropolitan Area Network Conecta redes dentro de uma cidade.
WAN Wide Area Network Redes de longa distância (a Internet é a maior WAN).

3. Topologias de Rede

A topologia define a "forma" ou o layout da rede. As mais comuns hoje são:

  • Estrela: Todos os dispositivos se conectam a um nó central (geralmente um switch). Se um cabo falha, apenas aquele dispositivo cai.
  • Malha (Mesh): Todos se conectam com todos. É a mais resiliente, usada em sistemas críticos e roteadores Wi-Fi modernos.

4. O Modelo OSI: A Linguagem Universal

Para que dispositivos de fabricantes diferentes se comuniquem, o modelo OSI (Open Systems Interconnection) divide o processo em 7 camadas. Pense nisso como uma linha de montagem:

1. Física: Transmissão de bits brutos pelo meio físico.

2. Enlace: Endereçamento físico (MAC Address) e detecção de erros.

3. Rede: Roteamento de pacotes e endereçamento lógico (IP).

4. Transporte: Controle de fluxo e correção de erros (TCP/UDP).

5. Sessão: Estabelecimento e manutenção de conexões.

6. Apresentação: Tradução, criptografia e compressão de dados.

7. Aplicação: Onde o usuário interage (HTTP, FTP, DNS).

5. Protocolos Essenciais: Os "Atores" da Internet

Protocolos são regras que regem a comunicação. Sem eles, os computadores falariam "línguas" diferentes.

  • IP (Internet Protocol): Identifica o endereço de cada máquina.
  • TCP (Transmission Control Protocol): Garante que os dados cheguem na ordem correta e sem perdas.
  • NS (Domain Name System): O "catálogo telefônico" da web. Ele transforma google.com no endereço IP numérico correspondente.
  • HTTP/HTTPS: Protocolo de transferência de hipertexto usado para carregar sites.

Exercícios de Fixação: Redes de Computadores

1. Uma empresa instalou uma rede interna para conectar os computadores de todos os seus funcionários dentro de um único edifício de escritórios. Essa configuração é classificada como uma rede:

  • A) PAN (Personal Area Network)
  • B) LAN (Local Area Network)
  • C) MAN (Metropolitan Area Network)
  • D) WAN (Wide Area Network)

2. No Modelo OSI, qual camada é responsável por determinar a melhor rota para os dados e gerenciar o endereçamento lógico (IP) dos pacotes?

  • A) Camada 1 - Física
  • B) Camada 2 - Enlace
  • C) Camada 3 - Rede
  • D) Camada 4 - Transporte

3. Quando você digita "www.google.com" no navegador, um serviço entra em ação para traduzir esse nome amigável para o endereço IP real do servidor. Esse protocolo é o:

  • A) HTTP
  • B) TCP
  • C) DNS
  • D) FTP
